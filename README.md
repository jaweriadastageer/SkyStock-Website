A modern web application combining a Weather App and a Finance App into a single platform. 
Built with modern frontend technologies, it features a clean, responsive UI and fetches real-time 
data through APIs. This project demonstrates skills in API integration, frontend development, 
and data handling, making it ideal for portfolio showcase or learning purposes.
